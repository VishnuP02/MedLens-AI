# MedLens AI

MedLens AI is an AI-powered healthcare application that helps patients better understand medical reports by translating complex medical terminology into patient-friendly explanations.

## Features

- Upload medical reports
- AI-generated plain language summaries
- Highlight important findings
- Suggested questions for healthcare providers
- Secure document processing

## Tech Stack

### Frontend
- React
- Tailwind CSS

### Backend
- FastAPI
- Python

### AI
- OpenAI API

### Document Processing
- pdfplumber
